## About PyWEED

PyWEED is an application for retrieving event-based seismic data.

PyWEED works on Windows, Mac, and Linux (64-bit). It is built on [ObsPy](https://obspy.org/) and
[PyQt](https://pypi.python.org/pypi/PyQt4).
